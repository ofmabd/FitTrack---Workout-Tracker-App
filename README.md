FitTrack is a full-stack workout tracking web app built with Flask and SQLite.  
Users can log workouts, add exercises with sets/reps/weight, and view progress over time.

## Features

- Log workouts with notes and duration
- Add multiple exercises per workout
- View workout history and progress
- Delete workouts
- Clean responsive UI

## Tech Stack

- Python (Flask)
- CS50 SQL (SQLite)
- HTML / CSS
- Jinja Templates

## Installation

Clone the repo and install dependencies:

pip install -r requirements.txt

Run the app:

python -m flask run --no-reload

Open:

http://127.0.0.1:5000
